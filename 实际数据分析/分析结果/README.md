# Real-data Analysis Outputs

This folder stores protected summary outputs from the gait-video real-data
analysis.

The raw gait videos and individual-level feature tables are not included in this
repository.  The encrypted archive below contains only the summary tables used
to report prediction performance, selected sources, and regression coefficients:

```text
加密表格/real_data_summary_tables.tar.gz.enc
```

The archive is encrypted with AES-256-CBC through OpenSSL.  The password is not
stored in the repository and should be shared separately only when access is
approved.
